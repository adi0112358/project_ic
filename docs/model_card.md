# Model Card – Project IC (Synthetic Demo)

## Intended use
This project is a **college‑level prototype** for demonstrating an AI pipeline that combines lab reports and skin images. It is **not** a clinical tool and must not be used for diagnosis or treatment.

## Data
- **Tabular data**: synthetic lab values generated by a script.
- **Images**: synthetic lesion patterns generated by a script.

No real patient data was used.

## Models
- Tabular models: logistic regression classifiers for
  - `fungal_infection`
  - `diabetes_status`
  - `severity`
  - `fungal_type`
- Image model: logistic regression over simple color/edge features

## Performance
Metrics are near‑perfect on synthetic data. This is expected and does **not** reflect real‑world performance.

## Limitations
- Trained entirely on synthetic data.
- Not validated clinically.
- Not calibrated for real patient distributions.
- No safety guarantees for decision support.

## Ethical and safety notes
- Do **not** present outputs as medical advice.
- Always defer to clinician evaluation.
- Clearly state limitations in demos and reports.
